Connect to kafka console through docker. 
cd usr/bin to find the following scripts

See group ids: kafka-consumer-groups.sh --bootstrap-server kafka:9092 --list
See kafka topics: kafka-topics.sh --list --bootstrap-server localhost:2181

Console generator to test: 
kafka-console-producer.sh --broker-list kafka:9092 --topic productions

Read kafka topics: 
kafka-console-consumer.sh --topic consume --from-beginning --bootstrap-server kafka:9092

how to deploy working environment:
git clone https://github.com/iamnives/docker-sp
cd .\docker-sp\docker-compose\
docker-compose up
http://localhost:9390/editor

how to publish data to kafka topic from docker (make sure to put the csv files under docker-compose/kafka-data/):
Click "CLI" button on docker UI or use "docker exec -it kafka /bin/bash" in command line
cd usr/bin
kafka-console-producer.sh --bootstrap-server kafka:9092 --topic productions < kafka-data/sample_data.csv
